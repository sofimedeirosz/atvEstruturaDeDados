# atvEstruturaDeDados
Atividade de Estrutura de Dados sobre Listas

Colaboradores:
- Sofia Medeiros da Fonseca
- Gabriel Moreno
- João Gabriel Ohara
- Maria Eduarda de Souza Silveira

Funções Adicionadas:
- Remover todas ocorrencias e retornar quantos removeu
- Retornar qtd total de elementos válidos (exceto null)
- Adicionar múltiplo elementos
- Retornar o elemento armazenado numa determinada posição da lista
- Inserir um elemento numa posição específica da lista
- Remover um elemento numa posição específica da lista
- Remover todos os elementos da lista, deixando-a vazia
- Retornar o índice da última ocorrência de um elemento na lista
- Contar quantas vezes um determinado elemento aparece na lista
- Substituir todas as ocorrências de um elemento antigo por um novo elemento
